# Biblioteca-Virtual
Este projeto foi desenvolvido como parte integrante da disciplina de Tecnologia da Informação do 1º Semestre. A ideia central é uma biblioteca virtual personalizada que abriga uma curadoria selecionada dos meus 12 livros e mangás favoritos.

O diferencial da plataforma é a sua estética vintage, pensada para resgatar a nostalgia do papel envelhecido e o conforto de uma leitura acompanhada de um café, fugindo do minimalismo frio das interfaces modernas.
